Inspector CLI Installer

Install the latest Inspector CLI bundle with a single command. The installer pulls the current signed binary release, verifies the checksum, and installs it into your local user directory.

Install curl -fsSL https://cli.inspectorai.pro/install.sh | bash
Open install.sh View Releases

What it installs

The installer downloads the latest Inspector bundle, places it in ~/.local/share/inspector, and links the CLI into ~/.local/bin/inspector.

Update path

Once installed, use inspector update to check for and install new bundle releases from the same release channel.
