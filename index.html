Install Inspector CLI
in one command

Pulls the current signed binary release, verifies the checksum, and installs it into your local user directory.

Install
curl -fsSL https://cli.inspectorai.pro/install.sh | bash
Open install.sh View Releases
📦

What it installs

Downloads the latest Inspector bundle, places it in ~/.local/share/inspector, and links the CLI into ~/.local/bin/inspector.

🔄

Update path

Once installed, run inspector update to check for and install new bundle releases from the same release channel.

Published from the Inspector binary distribution repository via GitHub Pages.
